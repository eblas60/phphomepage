Explication
-----------

Cette application permet de générer une page de démarrage pour les navigateur internet

Install
-------

1.Créer une base de donnée
2.Configurer le fichier "config.inc.php3" avec les paramètres de votre table
3.Ouvrez la page index.php3 dans un navigateur
4.Les tables sont générée lors du premier lancement
6.Vous pouvez vous amuser avec l'application

Utilisation
-----------

7.Une fois que vous avez affiché la page de démarrage
8.Faites en votre page de démarrage par défaut (vous pouvez l'enregistrer en local si vous voulez)
